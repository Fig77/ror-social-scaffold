## Getting Started

To get a local copy up and running follow these simple example steps.

---

### Prerequisites

Ruby: 2.6.5
Rails: 6.0.2.2
sqlite3: >=1.4.2

---

### Setup

Instal gems with:

```
bundle install
```

Setup database with:

```
   rails db:create
   rails db:migrate
```

---

### Usage

Start server with:

```
    rails server
```

Open `http://localhost:3000/` in your browser.

---

### Run tests

```
    rails test
```
